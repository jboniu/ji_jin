# 支付宝基金自动分析系统 设计概要

## 核心功能
1. 抓取财经新闻与市场信息，作为分析输入。
2. 调用 AI 生成基金视角的市场分析报告。
3. 将分析结果保存为本地 Markdown 报告，便于查看和归档。
4. 支持定时执行，先实现每日晚间分析，再扩展到每周总结。
5. 预留持仓输入能力，后续可做个性化风险和仓位参考。

## 主要文件
- `fetch_news.py` - 获取新闻数据
- `analyze_fund.py` - 调用 AI 生成基金分析
- `generate_report.py` - 组装流程并输出 Markdown 报告
- `reports/` - 保存生成的分析报告
- `.env.example` - 环境变量示例
- `requirements.txt` - 项目依赖

## 技术选型
- Python
- RSS / 公开财经信息源
- OpenAI API
- Markdown 作为首个报告格式
- Windows 任务计划程序作为后续定时方案

## 第一阶段范围
1. 手动运行脚本，可生成一份 Markdown 报告。
2. 报告包含市场状态、基金类型影响、操作参考、风险提示、后续关注点。
3. 先不做邮件推送、可视化页面和复杂持仓联动。

## 原子功能点拆分
1. 初始化项目骨架和依赖文件。
2. 实现新闻抓取并能本地看到抓取结果。
3. 实现 AI 分析并能输出结构化文本。
4. 实现 Markdown 报告落盘。
5. 接入定时执行说明或脚本。
