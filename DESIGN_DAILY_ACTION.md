# 日报操作观察版 设计概要

## 核心功能
1. 在现有日报基础上，增加近一周基金走向回顾。
2. 结合当天新闻和用户持仓，输出更贴近盘中决策的观察建议。
3. 报告中增加“今日 15:00 前重点关注”板块。
4. 对用户持仓给出加仓观察、减仓观察、继续持有观察的分组。
5. 保留现有新闻抓取、多用户发送、Markdown 存档和日志能力。

## 主要文件
- `analyze_fund.py` - 更新日报分析提示词和输出结构
- `generate_report.py` - 保持主流程不变，承接新报告结构
- `DESIGN_DAILY_ACTION.md` - 当前设计文档

## 技术选型
- 继续使用新闻驱动的 AI 分析
- 以持仓结构和最近一周走势回顾为主
- Markdown 继续作为日报输出格式

## 第一阶段范围
1. 不直接给“必须买卖”的绝对指令。
2. 改成“加仓观察 / 减仓观察 / 持有观察”。
3. 增加“近一周回顾”和“今日 15:00 前重点关注”。
4. 暂不接入实时净值、实时盘口或基金估值接口。

## 原子功能点拆分
1. 更新日报分析提示词。
2. 更新报告输出结构。
3. 让报告更贴近用户单只持仓。
4. 更新文档说明。
