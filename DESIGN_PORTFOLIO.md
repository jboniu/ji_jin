# 持仓个性化分析 设计概要

## 核心功能
1. 支持用户维护自己的基金持仓结构输入。
2. 在生成报告时，将持仓信息一起传给 AI，输出更贴近个人配置的分析。
3. 报告中增加“我的持仓影响”和“个性化风险提醒”部分。
4. 保留现有新闻抓取、Markdown 保存、邮件发送和定时任务能力。
5. 后续可扩展为周报、月报和资产再平衡建议。

## 主要文件
- `portfolio.json` - 用户持仓配置
- `portfolio.py` - 读取和格式化持仓信息
- `analyze_fund.py` - 将新闻和持仓一起交给 AI 分析
- `generate_report.py` - 生成包含个性化内容的报告
- `DESIGN_PORTFOLIO.md` - 当前设计文档

## 技术选型
- JSON 作为首版持仓输入格式
- Python 读取和校验持仓配置
- 兼容 OpenAI 接口的大模型做个性化分析
- Markdown 继续作为报告输出格式

## 第一阶段范围
1. 用户手动维护 `portfolio.json`。
2. 系统读取持仓结构并参与报告生成。
3. 报告增加个人持仓相关分析内容。
4. 暂不做图表、数据库和自动同步支付宝持仓。

## 原子功能点拆分
1. 增加持仓配置文件和读取逻辑。
2. 在本地输出可验证的持仓摘要。
3. 将持仓摘要接入 AI 分析提示词。
4. 在报告中新增个性化分析板块。
5. 更新文档和配置说明。
